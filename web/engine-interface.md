# Engine Interface

Friendly to web developers. 开发人员无需3D视觉和图形学知识，接口设计遵从广大前端工程师开发习惯

* Cartesian -&gt; earth
* 3D elements  -&gt; HTML elements
* rendering effects -&gt; CSS style
* Interactions -&gt; Events

3D elements:

* LOD Mesh
* LOD Point cloud
* BillBoard
* PolyLine, Polygon, Extrude volume
* Particles
* etc.
